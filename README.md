# 🏭 GOODS FACTORY

**굿즈 기획 · 제작 원스톱 웹앱** — 기획부터 시장분석·굿즈선정·디자인·목업·스케줄·발주까지.

굿즈(팬시/캐릭터/게임/브랜드 상품)를 만들 때 필요한 흐름을 한 곳에서 처리하는 도구입니다. 설치 없이 브라우저에서 바로 쓸 수 있고, 모든 데이터는 **내 브라우저에만 저장**됩니다(서버·로그인 없음).

## ✨ 주요 기능

- **7단계 흐름** — 기획 → 시장 분석 → 굿즈 선정 → 디자인 → 목업 → 스케줄 → 산출물
- **굿즈 카탈로그 45종** — 품목별 제작단가·시장가·최소수량·제작기간(참고치) 내장
- **마진·손익분기 자동 계산** — 국내/중국 제작지별 단가·납기 반영
- **차별화 레이어** — 2026 트렌드 + 포화도(흔한 품목 경고) + 굿즈별 차별화 포인트
- **굿즈 목업** — 디자인을 8개 제품(티셔츠·머그·키링·에코백 등)에 얹어 미리보기
- **제작 스케줄** — 굿즈별 단계 일정(간트) + 완료 예상일
- **산출물 문서** — 기획안 · 디자인제안+견적 · 견적서 · 스케줄표 · 발주서 · 상세페이지 (앱 안에서 보며 수정, HTML→PDF 저장)
- **제안서 5가지 디자인 양식** — 기본 / 키즈 / 러블리 / 럭셔리 / 다크
- **여러 프로젝트 관리** — 프로젝트별 저장·열기·복제
- **AI 프롬프트 생성** — 각 단계 프롬프트를 복사해 챗GPT·제미나이·클로드에 사용 (AI 없이도 전 기능 사용 가능)

## 🚀 사용 방법

1. **바로 쓰기**: https://nextsunny-ai.github.io/goods-factory/ 접속, 또는 `index.html`을 브라우저로 열기
2. 왼쪽 위 단계 탭을 따라 입력
3. 마지막 "산출물" 단계에서 문서를 열어 보고 저장

> 데이터는 이 브라우저에만 저장됩니다. 다른 기기로 옮기려면 우측 상단 ⋯ 메뉴 → "이 프로젝트 파일로 저장(.json)".

## 🖥️ 로컬 실행

```bash
# 이 폴더에서
python -m http.server 8080
# 브라우저에서 http://localhost:8080
```

또는 `index.html`을 더블클릭해도 동작합니다.

## 🔒 개인정보

- 서버·계정·로그인 없음. 입력한 모든 내용은 브라우저 로컬 저장소에만 남습니다.
- AI 연결(선택) 시 입력한 키도 브라우저에만 저장됩니다.

순수 HTML/CSS/JavaScript (프레임워크·빌드 없음). 어떤 정적 호스팅에서도 동작합니다.
